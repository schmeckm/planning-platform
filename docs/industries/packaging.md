# Packaging Pack

> 🚧 **In development** — contributions welcome. See [Build Your Own Pack](/industries/custom).

## Planned Constraints

- `packaging.changeover-matrix` — sequence-dependent changeover times between SKUs
- `packaging.allergen-clearance` — allergen-containing SKUs require line clearance before allergen-free production
- `packaging.line-speed` — dynamic throughput based on format and material type
- `packaging.minimum-run-length` — minimum batch size per format change (waste reduction)
