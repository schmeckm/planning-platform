# Release-Notes-Konvention

> Diese Seite wird gerade übersetzt. [Englische Version lesen](/conventions/release-notes)

Kurzfassung:

- Release Notes sind **lesbare Geschichten** für Anwender; das [Changelog](/community/changelog) bleibt die technische Quelle.
- Titel: `<version>: <prägnanter Untertitel>`
- Pfad: `/docs/community/release-notes/<version>.md`
- Karten-Registry: `/docs/.vitepress/data/releases.ts`
- Vorlage und Checkliste: [Release Notes Convention (EN)](/conventions/release-notes)
