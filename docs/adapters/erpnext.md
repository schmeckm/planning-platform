# ERPNext Adapter

> 📋 **Planned** — contributions welcome. See [Build an Adapter](/adapters/custom).

ERPNext is a popular open-source ERP. This adapter will map Work Orders, BOM operations, and warehousing data into the PCP canonical model — making PCP a natural companion to ERPNext for manufacturing planning.

Want to lead this adapter? Open a GitHub Discussion tagged `adapter:erpnext`.
